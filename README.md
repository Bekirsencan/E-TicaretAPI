# ECommerce Book Store

A simple ecommerce store for selling books.

# Setup

1. Create venv
```
py -m venv venv
```
2. Activate venv
```
venv\Scripts\activate
```
3. Install requirements.txt
```
pip install -r requirements.txt
```
